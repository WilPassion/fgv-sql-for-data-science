# SQL - Structured Query Language — FGV 📊

This repository contains my practical scripts and database developments from the **SQL - Structured Query Language** course at **Fundação Getulio Vargas (FGV)**, taught by *Antonio Sergio Ferreira Bonato*. 

I am developing this project to enhance my hard skills in data analytics, focusing on leveraging production-ready SQL to manipulate, query, and draw insights from structured datasets.

---

## 🎯 Course Objectives Developed

*   **Querying & Summarizing:** Query and summarize data within relational tables.
*   **Large-Scale Analysis:** Analyze large datasets efficiently using optimized logic.
*   **Database Schema Management:** Create, alter, and drop tables to build robust structures capable of storing varying data volumes (from small to large scale).
*   **Data Manipulation (DML):** Insert new records into tables, update existing data, and purge obsolete records.

## 🛠️ Tech Stack & Key Concepts Applied

*   **Primary Database Engine:** PostgreSQL (All scripts strictly follow Postgres syntax guidelines).
*   **Data Source Integration:** SQLite (`.db` source file migrated/connected to PostgreSQL).
*   **Database Client:** DBeaver Community Edition.
